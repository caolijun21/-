# 管道智能巡检系统移动端应用

## 项目简介

本项目是管道智能巡检系统的移动端Web应用，旨在与树莓派上的管道智能巡检系统 v4.3 进行交互，提供便捷的远程操控、实时监控、数据查看和报告管理功能。

## 功能特性

- **实时视频监控**：显示MJPEG视频流，支持全屏模式和截图功能
- **手动控制**：虚拟摇杆控制小车移动，支持速度调节和急停功能
- **状态监控**：实时显示系统状态，包括模式、IP、温度、里程、缺陷计数等
- **任务管理**：开始/结束巡检任务，记录操作员、起止时间、总距离
- **缺陷记录查看**：查看历史缺陷列表，包含图像、类型、置信度、距离、时间戳
- **报告生成与管理**：触发报告生成（TXT/JSON/PDF），查看已生成报告列表，下载报告
- **网络设置**：扫描WiFi，连接新网络，切换热点/客户端模式
- **系统设置**：配置邮箱参数，重置里程，拍照（手动抓图）
- **数据同步**：离线时缓存数据，连接后自动上传

## 技术栈

- **前端框架**：React + Vite
- **状态管理**：Redux Toolkit
- **样式**：自定义CSS
- **通信**：HTTP REST API + WebSocket
- **PWA**：支持离线访问和添加到主屏幕

## 使用说明

1. **连接设备**：在设置页面输入树莓派的IP地址和端口（默认5000），点击保存
2. **实时监控**：在主页查看实时视频流，点击视频区域可全屏观看
3. **手动控制**：使用虚拟摇杆控制小车移动，调节速度滑块控制速度，点击急停按钮立即停止
4. **任务管理**：点击开始任务按钮，输入操作员姓名，开始巡检任务；任务完成后点击结束任务按钮
5. **缺陷记录**：在缺陷页面查看历史缺陷记录，点击缺陷卡片查看详情，可删除缺陷记录
6. **报告管理**：在报告页面选择报告格式，点击生成按钮生成报告；查看已生成报告列表，点击下载按钮下载报告
7. **网络设置**：在设置页面扫描WiFi，选择网络并输入密码连接；点击启动热点按钮切换到热点模式
8. **系统设置**：在设置页面配置邮箱参数，点击测试连接按钮测试邮箱连接；点击重置里程按钮重置里程；点击拍照按钮手动拍照

## PWA功能

本应用支持PWA（Progressive Web App）功能，可以添加到主屏幕，离线访问。

### 添加到主屏幕

1. 在Chrome浏览器中打开应用
2. 点击浏览器菜单（三个点）
3. 选择"添加到主屏幕"
4. 按照提示完成安装

## 注意事项

- 应用需要与树莓派上的管道智能巡检系统 v4.3 配合使用
- 确保手机与树莓派在同一网络环境中
- 首次使用时需要在设置页面配置树莓派的IP地址和端口
- 视频流可能会消耗较多网络带宽，请确保网络连接稳定
